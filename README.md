# NuxtShop

Интернет-магазин на Nuxt 3.

## Установка

```bash
npm install
```

## Разработка

Запустить dev-сервер на `http://localhost:3001`:

```bash
npm run dev
```

### Storybook

Запустить Storybook на `http://localhost:6006` (в отдельном терминале):

```bash
npm run storybook
```

> Для просмотра Storybook в Nuxt DevTools запусти оба процесса одновременно и открой DevTools (`Shift + Alt + D`).

## Сборка

```bash
npm run build
```

Превью продакшн-сборки:

```bash
npm run preview
```
